<h2>File Transfer App</h2>

<h3>Project Description</h3>
<p>The File Transfer Application is a simple GUI tool that automates transferring files from one directory to another. It specifically moves files modified within the last 24 hours, ensuring only recent files are transferred.</p>

<h3>Features</h3>
<ul>Source Directory Selection: Choose the directory containing the files to be transferred.</ul>
<ul>Destination Directory Selection: Choose the destination directory for the files.</ul>
<ul>File Transfer: Automatically transfers files modified in the last 24 hours.</ul>
<ul>Exit Button: Easily close the application.</ul>

<h3>Technologies and Skills Used</h3>
<ul>Python: Main programming language.</ul>
<ul>Tkinter: For creating the graphical user interface.</ul>
<ul>OS Module: For handling file paths and directory contents.</ul>
<ul>Shutil Module: For moving files.</ul>
<ul>Datetime Module: For filtering files by modification time.</ul>

<h3>How to Use</h3>
<ol>Select Source Directory: Click "Select Source" to choose the source directory.</ol>
<ol>Select Destination Directory: Click "Select Destination" to choose the destination directory.</ol>
<ol>Transfer Files: Click "Transfer Files" to move recently modified files.</ol>
<ol>Exit Application: Click "Exit" to close the application.</ol>
<br>
<p>This project showcases the use of Python and Tkinter for creating a functional file management tool with a graphical interface.</p>




<h2>Django Hotel Website w/ Room Service Menu</h2>

<h3>Project Description</h3>
<p>This project is a Django-based website for a hotel that includes a food menu management system. It allows users to view, add, update, and delete food menu items through an admin interface.</p>

<h3>Features</h3>
<ul>Admin Console: View all food menu items.</ul>
<ul>Detail View: View and edit details of individual menu items.</ul>
<ul>Delete Functionality: Confirm and delete menu items.</ul>
<ul>Create Record: Add new items to the food menu.</ul>

<h3>Technologies and Skills Used</h3>
<ul>Django: Used for creating models, views, and templates to manage and display food menu items.</ul>
<ul>Python: Implemented backend logic for handling CRUD operations and form submissions.</ul>
<ul>HTML/CSS: Designed the frontend interface for a user-friendly experience in managing menu items.</ul>
<ul>SQLite: Used as the database to store food menu items and related data during development and testing.</ul>
<ul>Forms: Utilized Django Forms to handle input validation and rendering in templates.</ul>

<h3>How to Use</h3>
<ol>Admin Console: Navigate to the admin console to view all menu items.</ol>
<ol>Detail View: Click on a menu item to view and edit its details.</ol>
<ol>Delete Functionality: Confirm and delete items from the menu.</ol>
<ol>Create Record: Add new items through the create record form.</ol>
<br>
<p>This project demonstrates the use of Django for building a functional web application with CRUD (Create, Read, Update, Delete) operations, providing a practical example of web development and database management.</p>


<h2>Django Checkbook Banking App</h2>

<h3>Project Description</h3>

<ul>Home Page: View a form to select an account and view its balance.</ul>
<ul>Create New Account: Add new bank accounts.</ul>
<ul>Balance Sheet: View the balance and transaction history of a selected account.</ul>
<ul>Add Transaction: Add deposits or withdrawals to an account.</ul>

<h3>Technologies and Skills Used</h3>
<ul>Django: Used to create models, views, and templates for account and transaction management.</ul>
<ul>Python: Implemented backend logic for form handling and database interactions.</ul>
<ul>HTML/CSS: Designed the frontend interface to ensure a user-friendly experience.</ul>
<ul>SQLite: Used as the database to store account and transaction information.</ul>
<ul>Forms: Utilized Django Forms to handle input validation and rendering in templates.</ul>
<ul>ORM (Object-Relational Mapping): Employed Django's ORM to interact with the database and handle CRUD operations.</ul>

<h3>How to Use</h3>
<ol>Home Page: Select an account from the form to view its balance.</ol>
<ol>Create New Account: Navigate to the "Create New Account" page to add a new account.</ol>
<ol>Balance Sheet: View the balance and transaction history of a selected account.</ol>
<ol>Add Transaction: Use the "Add Transaction" page to record deposits or withdrawals</ol>
<br>
<p>This project showcases the use of Django for building a comprehensive web application with functionality for managing financial accounts and transactions.</p>



<h2>Student Tracker Application</h2>

<h3>Project Description</h3>
<p>This project is a Student Tracker application developed using Python's Tkinter library for the graphical user interface (GUI). It demonstrates object-oriented programming (OOP) principles, database management, and Tkinter's parent-child relationships for an effective and user-friendly student management system.</p>

<h3>Features</h3>
<ul>Add Student: Create and add new student records.</ul>
<ul>Update Student: Edit existing student information.</ul>
<ul>Delete Student: Remove student records from the database.</ul>
<ul>View Students: Display student information in a list.</ul>

<h3>Technologies and Skills Used</h3>
<ul>Python: Implemented backend logic and database operations.</ul>
<ul>Tkinter: Used for creating the GUI, managing user inputs, and displaying data. Grid Geometry used to manage layout of widgets for a structured and user-friendly interface, and Created interactive elements for user actions through menu and button features.</ul>
<ul>SQLite: Managed the database to store and retrieve student records. Also implemented CRUD Operations and Data Validation.</ul>
<ul>OOP Principles: Organized code into classes and methods for modularity and reusability.</ul>
<ul>File Handling: Managed application settings and data persistence.</ul>
<ul>Event Handling: Captured and responded to user actions in the GUI.</ul>

<h3>How to Use</h3>
<ol>Add Student: Enter student details in the form fields and click 'Submit' to add a new student.</ol>
<ol>Update Student: Select a student from the list, modify the details, and click 'Update' to save changes.</ol>
<ol>Delete Student: Select a student from the list and click 'Delete' to remove the student from the database.</ol>
<ol>Close Application: Click 'Close' to exit the application.</ol>
<br>
<p>This project highlights the use of Tkinter for building a robust desktop application, combined with SQLite for effective data management, demonstrating practical skills in Python programming and GUI development.</p>


<h2>Mythos: Text Based Adventure Game</h2>

<h3>Project Description</h3>
<p>This project is a text-based adventure game developed using Python. The game guides players through a mythical journey where their decisions affect the outcome. Players can choose between interacting nicely or meanly with various characters they encounter, leading to different endings based on their choices.</p>

<h3>Features</h3>
<ul>Interactive Storyline: Players make choices that influence the story's direction and ending.</ul>
<ul>Variable Tracking: Tracks player interactions and decisions, influencing the game outcome.</ul>
<ul>Multiple Endings: The game has different endings based on the player's decisions.</ul>

<h3>Technologies and Skills Used</h3>
<ul>Python: Developed the entire game logic and flow.</ul>
    <ul>Functions and Variables: Managed game states, player interactions, and outcomes using functions and variables.</ul>
    <ul>Conditional Statements: Created interactive elements by using if-else conditions to determine the game's direction based on player choices.</ul>
    <ul>Input/Output: Engaged the player by taking inputs and providing outputs to guide the game narrative.</ul>
<ul>Loops: Used loops to manage the game flow and replayability.</ul>
<ul>String Formatting: Displayed dynamic messages based on player input and game state.</ul>

<h3>How to Use</h3>
<ol>Start the Game: Run the game script to start the adventure.</ol>
<ol>Make Decisions: Follow the prompts to make decisions by typing the choices provided.</ol>
<ol>Track Progress: The game tracks your interactions and adjusts the storyline accordingly.</ol>
<ol>Reach an Ending: Based on your decisions, reach one of the possible endings.</ol>
<ol>Play Again: Choose to play again and try different choices to see how the story changes.</ol>


<br>
<p>This text-based adventure game demonstrates practical skills in Python programming, including function calls, variable management, and interactive storytelling.</p>
